All API Endpoints

Full list of available routes. 🔒 means a valid JWT is required in the Authorization header.


POST /auth/register Create a new user account
POST /auth/login Authenticate & receive JWT tokens
POST /auth/refresh-token Generate new access token via refresh
GET /auth/profile Get user profile 🔒
GET /auth/login-history View login activity logs 🔒
POST /auth/logout Invalidate current session 🔒
