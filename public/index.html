Authentication API

Production-ready JWT & OAuth2 service featuring robust token rotation, rate limiting, and RBAC.

Node.js Express MongoDB JWT Bcrypt
System Online & Ready

Available Endpoints

POST /auth/register Create a new user account
POST /auth/login Authenticate & receive JWT tokens
GET /auth/profile Retrieve protected user details
POST /auth/refresh-token Generate new access token via refresh token
GET /auth/login-history View security and login event logs
POST /auth/logout Invalidate current sessions
